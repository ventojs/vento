# Framework integrations

Vento can be used with the following frameworks:

- Lume: [supported by default](https://lume.land/plugins/vento/)
- Eleventy:
  [eleventy-plugin-vento](https://www.npmjs.com/package/eleventy-plugin-vento).<br>
  Check out the [meetup video where vrugtehagel introduces Vento](https://www.youtube.com/watch?v=_854y7c0D-0) to the 11ty community.
- Express: [express-vento](https://www.npmjs.com/package/express-vento)
